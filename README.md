Data Jobs Dashboard

This project is a Power BI dashboard that explores trends in data-related jobs, including demand, salaries, and job roles.

Objective

To analyze the data job market and identify patterns in job demand, salary distribution, and role popularity.

Tools Used
Microsoft Power BI
Data visualization

Dashboard Overview

This dashboard includes:

Total number of job listings
Average yearly and hourly salaries
Job demand over time
Comparison between salary and roles
Drill-through functionality for deeper analysis
